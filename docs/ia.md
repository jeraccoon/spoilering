# Uso de IA en Spoilering

## Objetivo

Reducir el esfuerzo de creación de contenido y facilitar la colaboración.

---

## Funcionalidades

La IA permite:

* convertir texto libre en contenido estructurado
* generar resúmenes
* identificar personajes
* completar secciones

---

## Flujo

1. Usuario introduce texto libre
2. IA procesa el contenido
3. Se genera una estructura completa
4. Usuario revisa
5. Usuario guarda

---

## Reglas importantes

* la IA no publica automáticamente
* el usuario siempre revisa
* evitar invenciones cuando falte información
* priorizar claridad y estructura

---

## Ventajas

* menos fricción para colaborar
* contenido más homogéneo
* mayor velocidad de creación
